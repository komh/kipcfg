                                kipgcfg v0.1.1
                               ----------------
                            
1. 소개
-------
  
  이 프로그램은 DHCP 서버로부터 IP 주소를 자동 할당받는 프로그램입니다. 

2. 제작 환경
------------
  
  - 한국어판 OS/2 Warp v4 with FixPak #15
  
  - Open Watcom v1.8
      
3. 실행에 필요한 환경
---------------------

  - TCP/IP v4.3

4. 시험 환경
------------

  한국어판 OS/2 Warp v4 with FixPak #15 에서 시험했습니다.

5. 기능
--------

  - 여러 인터페이스 지원
  
  - IP 주소만 받기 지원 
    
6. 설치
-------

  kipcfg.exe 을 아무 디렉토리에 같이 넣어두시면 됩니다.
  
7. 사용법
---------

7.1. 실행하기 
-------------

    kipcfg /req if | /rel if | /iponly | /wait secs | /q

7.2. 선택사항
-------------
    if          인터페이스 이름, 예) lan0
    secs        최대 대기 시간, 1초부터 60초까지
    
    /req        해당 인터페이스의 IP 주소 받기
    /rel        해당 인터페이스의 IP 주소 반환하기
    /iponly     IP 주소만 받기(기본 게이트웨이와 DNS 서버가 필요없을 때)
    /wait       IP 주소를 받을 때까지 주어진 시간동안 기다리기
    /q          kipcfg 대몬을 끝내기
  
8. 제한 사항 또는 알려진 벌레
-----------------------------

  - DHCP 로부터 할당받은 IP 주소의 충돌 여부를 확인하지 않음
  
  - 172.xx.100.100 IP 주소를 임시로 사용함( xx 는 16 ~ 31 )
  
  - LOOPBACK 인터페이스가 활성화 됨
  
9. 할 것들...
-------------

  - ifconfig.exe 대신에 소켓 API 사용
  
  - 할당받은 IP 주소 충돌 확인하기
  
  - 임시 IP 주소 없이 통신하기
